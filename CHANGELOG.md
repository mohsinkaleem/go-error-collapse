# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-31

### Added
- Initial release
- Smart detection of Go error handling patterns
- Visual folding of error blocks using VS Code's native folding
- Auto-collapse on file open (configurable)
- Auto-collapse on file save (configurable)
- Transparency mode for error blocks
- Keyboard shortcut `Cmd/Ctrl+Shift+E` for quick toggle
- Configuration options for customization
- Support for custom error variable patterns
